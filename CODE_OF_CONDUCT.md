# Code of Conduct

## 約束

参加者全員にとって嫌がらせのない体験を提供することを約束します。年齢・体型・障害・
民族・性自認や性表現・経験の有無・国籍・容姿・人種・宗教・性的指向にかかわらず、
敬意あるコミュニティを目指します。

## 基準

- 歓迎的で包摂的な言葉を使う
- 異なる視点・経験を尊重する（Conclave 自体が「異質性の供給」を価値とします）
- 建設的な批評を穏やかに受け入れる
- コミュニティにとって最善のことに集中する

## 容認しない行為

- 性的な言動・表現、侮辱・中傷、個人攻撃、公私の嫌がらせ
- 許可のない個人情報（住所・連絡先等）の公開

## 執行

違反は GitHub Issue または所有者への連絡で報告できます。所有者は不適切な行為に対し、
コメントの編集・削除、コントリビューションの拒否、参加の一時/恒久的制限を行えます。

本ドキュメントは [Contributor Covenant](https://www.contributor-covenant.org) v2.1 を
簡約したものです。
