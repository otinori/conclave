# 撤退条件

Conclave のペルソナは支援ツールである。ユーザーの実際の仕事を阻害する、判断の幅を狭める、または独自の価値を提供しなくなったペルソナは、凍結または引退させる。

## レビュー頻度

Phase alpha の間は、稼働中のペルソナを週次でレビューする。

## 凍結トリガー

以下のいずれかが起きたペルソナは凍結する:

- 他のペルソナの見解を繰り返し重複している。
- 有用な判断材料を生まないまま作業負荷だけを増やしている。
- 判断を支援するのではなく、ユーザーを特定の決定へ誘導している。
- 記憶が弱い、あるいはレビューされていない蒸留から膨らんでいる。
- 運用が機密性やライセンスの前提と矛盾している。

## 引退トリガー

以下の場合はペルソナを引退させる:

- 複数回のレビューで同じ凍結トリガーが繰り返し現れる。
- そのペルソナの役割が、より適切に定義された別のペルソナに置き換えられた。
- 元のユースケースがもはや存在しない。
- そのペルソナの運用が、設定された作業負荷予算を超えて消費している。

引退は理由と（あれば）代替案とともに `.conclave/retro/` に記録する。
